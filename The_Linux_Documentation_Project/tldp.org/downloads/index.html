 Linux Documentation Project Downloads

Along with the many HOWTOs, Guides and other documents published by the LDP, we also develop tools and utilities for working with documentation. These tools are all available under the GPL.

db2omf

Db2omf is a conversion utility that reads a DocBook file (either XML or SGML), and writes out an OMF file. The OMF file is suitable for submission to a ScrollKeeper database. More information is available here at the LDP.

  • db2omf-0.5.tar.gz (11k)
  • db2omf-0.4.tar.gz (11k)

texi2db

Texi2db is a conversion utility that converts GNU Texinfo source files into DocBook XML. More information is available here at the LDP.

  • texi2db-0.5.tar.gz (25k)
  • texi2db-0.4.2.tar.gz (24k)
  • texi2db-0.4.1.tar.gz (24k)
  • texi2db-0.4.tar.gz (24k)
  • texi2db-0.3.tar.gz (23k)

wt2db

Wt2db is a conversion utility that converts WikiText source files into DocBook XML/SGML. More information is available here at the LDP.

  • wt2db-1.0.tar.gz (19k)
  • wt2db-0.6.tar.gz (20k)
  • wt2db-0.5.tar.gz (17k)
  • wt2db-0.4.tar.gz (14k)
  • wt2db-0.3.tar.gz (12k)
