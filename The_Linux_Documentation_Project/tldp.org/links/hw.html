Linux Systems and CPUs

  • CPU Review has reviews of various CPUs/chipsets and some other hardware, as well as Linux kernel compilation benchmarks.
  • Cyrix 6x86, 6x86L or 6x86MX CPUs information, patches, utilities and links.
  • Information about the Intel Pentium Processor invalid instruction, also known as the "f00f" bug. The problem is fixed in kernels 2.1.63 and 2.0.32.
  • The Linux Laptop Volunteer Support Database.
  • A Linux on PowerPC Support Petition, asking for support from PowerPC manufacturers.
  • The server-linux mailing list.
  • Supported Hardware lists: the Hardware-HOWTO, the Red Hat Hardware Compatibility Lists.
  • uClinux - Embedded Microcontroller Linux for processors without MMU (Memory Management Units).
  • The VAR Howto - Linux Computer Companies (Search the VAR Howto - Search for Linux Computer Companies or Add a VAR to the VAR Howto - Add a Linux Computer Company).


Networks

  • ISDN for Linux information.
  • Gigabit Ethernet Performace, for Linux and FreeBSD.
  • Five-Minute Linux PPP Instructions.
  • Tutorial for a linux dsl router with firewall.
  • Xircom Pocket Ethernet for Linux.


Disk Drives and Storage

  • Iomega Jaz & Linux information.
  • PC-DISK, has a huge database of disk drive information, such as jumper settings, layouts and other parameters.
  • RAID Solutions for Linux, has lots of information about RAID and Linux.
  • DSD - Data Storage Devices for Linux applications, sells Linux compatible tape and optical drives.


Peripherals

  • CD-Writer compatibility list.
  • Connectix QuickCam information.
  • LCDproc, a driver for Matrix Orbital 20x4 LCD displays.
  • Linux got me kicked out of Wal-Mart and mpegCam, software to make MPEG movies with a Quickcam.
  • The Linux Joystick Driver, has information and drivers for many joysticks.
  • Linux Parallel Port Home Page, has lots of information about connecting devices to your parallel port.
  • Linux on USB page talks about the latest status of the kernel stack that will support USB.
  • SANE (Scanner Access Now Easy).
  • The Shifting Soundscape, an article in PC Magazine Online, has some good information about audio hardware on PC systems.


Printers

  • A driver for the HP OfficeJet.
  • The Linux Printing site contains a number of resources to help you with your Linux Printing problems, including the Printing HOWTO, a support database, a project list, printer vendor scorecards, and more.


General Hardware Information

  • Build Your Own PC!, step by step instructions on building your own PC.
  • Asynchronous Circuits and Binary Valued Mathematics, Serban E. Vlad home page. Containing papers in asynchronous circuits and binary valued mathematics.
  • HardSeek, a search engine that you can use to find information by hardware manufacturer.
  • Linux Hardware Database, a repository of ratings of compatibility and performance of computer hardware components under Linux -  "a dynamic hardware compatibility FAQ".
  • Linux Hardware.net, has lots of information about Linux hardware, and the "Driverbase" database of device drivers.
  • PC-Clone UNIX Hardware Buyer's Guide.
  • System Optimization Information, a PC hardware information site. Some of it is very Windows-specific, but it still has a lot of useful information for Linux users.
  • Tom's Hardware Guide, has lots of good information about PC hardware.
  • TuxMobil - Linux On Mobile Computers, a lot of useful hands-on information about installing and running Linux, BSD, Solaris and other UniXes on laptops, PDAs, cell phones, portable music players, wearables and other mobile computer devices.
  • X Window System Terminals, describes how to convert low-end PCs into X Terminals.
